## Qu'est-ce que FreightZoneTracker ?

FreightZoneTracker est un explorateur mondial de navires maritimes. Sélectionnez l'une des 17 grandes régions maritimes pour voir les navires actifs, leurs pavillons, types, vitesses, caps et destinations — le tout alimenté par des données AIS réelles actualisées toutes les six heures.

Utilisez les outils de dessin sur la carte pour délimiter une zone personnalisée et filtrer les navires exactement là où vous le souhaitez. Cliquez sur n'importe quel marqueur pour voir les détails complets du navire.

## Qu'est-ce que l'AIS ?

Le **Système d'Identification Automatique (AIS)** est une technologie anti-collision maritime imposée par les règles SOLAS de l'OMI pour les navires de plus de 300 tonneaux de jauge brute en navigation internationale, ainsi que pour tous les navires à passagers. Les navires transmettent leur position, vitesse, cap, nom et destination toutes les quelques secondes par radio VHF.

## Sources de données

FreightZoneTracker utilise le réseau coopératif **[AISHub](https://www.aishub.net/)** — un agrégateur de données AIS gratuit et communautaire qui regroupe les signaux de récepteurs bénévoles à terre du monde entier. Les données sont actualisées automatiquement toutes les 6 heures via un workflow GitHub Actions programmé.

## Open Source

FreightZoneTracker est entièrement open source. L'outil CLI (`freightcli.py`) et ce site web sont disponibles sur [GitHub](https://github.com/aallbrig/freightzonetracker).
